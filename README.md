## Tasks

### As a client

1. Draft a project pitch identifying the target user, the specific problem, and the desired outcome.
2. Decide on your tech stack preference and whether the app requires AI integrations.

### As a developer

1. Browse the `techin510-marketplace` repository issues.
2. Identify up to 3 project pitches you want to build.
3. Pair up with a client/proposer.

## Deliverable

### As a client
- Post your pitch as a GitHub Issue in the `techin510-marketplace` repo using the provided `pitch-template.md`.
- Tag it with your stack preference, AI requirement, and complexity level if any.

### As a developer
- Respond to projects that you want to build (no more than 3 projects) on the `techin510-marketplace` issues.
- Pair up with a client/proposer.

## Constraint
- Each student's proposer and developer must be different people.
